Game - Connect 4

Connect 4 - это игра на JavaScript для друх игроков. Задача - выстроить 4 фишки одного цета (красные или жельтый) в ряд по вертикали, по горизонтали или по диагонали раньше другого игрока. Игроки кликают мышкой по колонке игрового поля для хода. Программа отслеживает победителей и оповещает о победителе.
